Key Concepts:

Reflexive: if (a, a) ∈ R, for every a ∈ A

Symmetric: if (a1, a2) ∈ R implies (a2, a1) ∈ R, for all a1, a2∈ A

Transitive: if (a1, a2) ∈ R and (a2, a3) ∈ R implies (a1, a3) ∈ R

Equivalence Relation: reflexive, symmetric and transitive

One-one Function: distinct elements have distinct images

Types of Functions

One to one Function (Injective): A function f : X → Y is defined to be one-one if the images of distinct elements of X under f are distinct.

Onto Function (Surjective): A function f: X → Y is said to be onto if every element of Y is the image of some element of X under f.

One-one and Onto Function (Bijective): A function f: X → Y is said to be one-one and onto if f is both one-one and onto.

Case Based Question:

Students visit a railway museum. An exhibit shows rail lines on a track. Let L be the set of all rail lines and R be the relation on L defined by R = {(l₁, l₂): l₁ is parallel to l₂}.

  2. Find whether R is transitive
  3. If one rail line is y = 3x + 2, find the set of rail lines related to it

Assessment:

  1. A function f: A→B defined as f(x) = 2x is bijective. If A = {1,2,3,4}, find B.

  3. Check if S = {(a,b): a ≤ b³} on real numbers is reflexive, symmetric or transitive.
  4. For f: A→B where A = R-{5}, B = R-{1}, f(x) = (x-3)/(x-5), show f is bijective.

Explain Inverse Trigonometric Functions

  • Definition: A function that reverses the effect of the trigonometric function (e.g., sin-1x, cos-1x, tan-1x).
  • Principal Value Branch: The range in which the inverse function gives a unique output.
    • sin-1x: [-π/2, π/2]
    • cos-1x: [0, π]
    • tan-1x: (-π/2, π/2)

Properties and Graphs

  • Key Properties:
    • sin-1(-x) = -sin-1x
    • cos-1(-x) = π - cos-1x
    • tan-1x + tan-1y = tan-1((x + y)/(1 - xy)) for xy < 1
  • Graphs: Plot and analyze the graphs of sin-1x, cos-1x, and tan-1x using GeoGebra.

Applications

  • Solve problems involving heights and distances.
  • Use inverse trigonometric functions to find angles in geometric shapes.

Assessment

  4. A tower stands vertically on the ground. From a point on the ground 30 meters away, the angle of elevation to the top of the tower is tan-1(2/5). Find the height of the tower.

Define Matrices

A matrix is a rectangular array of numbers arranged in rows and columns. For example:

\[ A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix} \]

Types of Matrices

  • Row Matrix: A matrix with only one row.
  • Column Matrix: A matrix with only one column.
  • Square Matrix: A matrix with an equal number of rows and columns.
  • Diagonal Matrix: A square matrix where non-diagonal elements are zero.
  • Identity Matrix: A diagonal matrix with all diagonal elements as 1.

Operations on Matrices

  • Addition and Subtraction: Performed on matrices of the same order by adding/subtracting corresponding elements.
  • Multiplication: Defined if the number of columns in the first matrix equals the number of rows in the second matrix.

Transpose and Symmetric Matrices

  • Transpose of a Matrix: Obtained by interchanging rows and columns.
  • Symmetric Matrix: A square matrix equal to its transpose, i.e., \( A = A^T \).

  3. For the matrices \( A = \begin{bmatrix} 2 & 3 \end{bmatrix} \) and \( B = \begin{bmatrix} 4 \\ 5 \end{bmatrix} \), find \( AB \) and \( BA \).
  4. If \( A = \begin{bmatrix} 1 & -1 \\ 2 & 3 \end{bmatrix} \), find \( A^T \) and verify whether \( A + A^T \) is symmetric.

Key Concepts:

1. Determinants and their Properties

  • For a 2x2 matrix \( A = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \), the determinant is \( \text{det}(A) = ad - bc \).
  • Properties:
    • \(\text{det}(AB) = \text{det}(A) \cdot \text{det}(B)\)
    • \(\text{det}(A^T) = \text{det}(A)\)
    • If two rows/columns are identical, \(\text{det}(A) = 0\).

2. Minors and Cofactors

  • Minor of an element \( a_{ij} \) is the determinant of the submatrix obtained by deleting the \( i^{th} \) row and \( j^{th} \) column.
  • Cofactor: \( C_{ij} = (-1)^{i+j} \cdot \text{Minor of } a_{ij} \).

3. Adjoint and Inverse of a Matrix

  • Adjoint of \( A \) is the transpose of the cofactor matrix.
  • Inverse: \( A^{-1} = \frac{\text{adj}(A)}{\text{det}(A)} \), provided \(\text{det}(A) \neq 0\).

4. Applications of Determinants

  • Solving linear equations using Cramer's Rule.
  • Calculating area of triangles using determinants.

  4. Solve the system using Cramer's Rule: \( 2x + 3y = 5 \) \( 4x - y = 3 \).

Explain Continuity

  • Continuity at a Point: A function f is continuous at x = a if:
    1. f(a) is defined.
    2. limx→a f(x) exists.
    3. limx→a f(x) = f(a).
  • Continuity on an Interval: A function is continuous on an interval if it is continuous at every point in the interval.

Explain Differentiability

  • A function f is differentiable at x = a if the derivative f'(a) exists. Graphically, the function has a non-vertical tangent line at that point.

Types of Discontinuities

  1. Removable Discontinuity: limx→a f(x) exists but f(a) is not defined or limx→a f(x) ≠ f(a).
  2. Jump Discontinuity: Left-hand and right-hand limits exist but are not equal.
  3. Infinite Discontinuity: The limit approaches infinity.

Derivatives of Composite Functions (Chain Rule)

If y = f(u) and u = g(x), then the derivative of y with respect to x is:

dy/dx = dy/du · du/dx

Mean Value Theorem

If f is continuous on [a, b] and differentiable on (a, b), then there exists a point c ∈ (a, b) such that:

f'(c) = (f(b) - f(a))/(b - a)
