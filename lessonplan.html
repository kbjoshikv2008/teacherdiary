Key Concepts:

Reflexive: if (a, a) ∈ R, for every a ∈ A

Symmetric: if (a1, a2) ∈ R implies (a2, a1) ∈ R, for all a1, a2∈ A

Transitive: if (a1, a2) ∈ R and (a2, a3) ∈ R implies (a1, a3) ∈ R

Equivalence Relation: reflexive, symmetric and transitive

One-one Function: distinct elements have distinct images

Types of Functions

One to one Function (Injective): A function f : X → Y is defined to be one-one if the images of distinct elements of X under f are distinct.

Onto Function (Surjective): A function f: X → Y is said to be onto if every element of Y is the image of some element of X under f.

One-one and Onto Function (Bijective): A function f: X → Y is said to be one-one and onto if f is both one-one and onto.

Case Based Question:

Students visit a railway museum. An exhibit shows rail lines on a track. Let L be the set of all rail lines and R be the relation on L defined by R = {(l₁, l₂): l₁ is parallel to l₂}.

  1. Find whether R is symmetric
  2. Find whether R is transitive
  3. If one rail line is y = 3x + 2, find the set of rail lines related to it

Assessment:

  1. A function f: A→B defined as f(x) = 2x is bijective. If A = {1,2,3,4}, find B.
  2. Show that R on N×N defined by (a,b)R(c,d) if ad(b+c) = bc(a+d) is equivalence.
  3. Check if S = {(a,b): a ≤ b³} on real numbers is reflexive, symmetric or transitive.
  4. For f: A→B where A = R-{5}, B = R-{1}, f(x) = (x-3)/(x-5), show f is bijective.

Explain Inverse Trigonometric Functions

  • Definition: A function that reverses the effect of the trigonometric function (e.g., sin-1x, cos-1x, tan-1x).
  • Principal Value Branch: The range in which the inverse function gives a unique output.
    • sin-1x: [-π/2, π/2]
    • cos-1x: [0, π]
    • tan-1x: (-π/2, π/2)

Properties and Graphs

  • Key Properties:
    • sin-1(-x) = -sin-1x
    • cos-1(-x) = π - cos-1x
    • tan-1x + tan-1y = tan-1((x + y)/(1 - xy)) for xy < 1
  • Graphs: Plot and analyze the graphs of sin-1x, cos-1x, and tan-1x using GeoGebra.

Applications

  • Solve problems involving heights and distances.
  • Use inverse trigonometric functions to find angles in geometric shapes.

Case-Based Question

A ladder leans against a wall, making an angle θ with the ground. If the foot of the ladder is 4 meters away from the wall and the ladder is 5 meters long, answer the following:

  1. Find θ using inverse trigonometric functions.
  2. If the ladder slides down such that the angle becomes φ = cos-1(0.6), find the new distance of the foot from the wall.

Assessment

  1. Find the principal value of sin-1(1/2).
  2. Prove that tan-12 + tan-13 = 3π/4.
  3. Solve for x: cos-1x = 2sin-1(1/2).
  4. A tower stands vertically on the ground. From a point on the ground 30 meters away, the angle of elevation to the top of the tower is tan-1(2/5). Find the height of the tower.
