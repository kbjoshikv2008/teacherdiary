Class Test: Relations and Functions

Class 12 Mathematics | 20 Marks | 40 Minutes

Instructions:

  • Answer all questions
  • Show all necessary working
  • Marks are indicated against each question

1. A function \( f: [-4, 4] \rightarrow [0, 4] \) is given by \( f(x) = \sqrt{16 - x^2} \). Show that f is an onto function but not a one-one function. Further, find all possible values of 'a' for which \( f(a) = \sqrt{7} \).

[5 marks]

2. Let R be the relation defined in the set A = {1, 2, 3, 4, 5, 6, 7} by \( R = \{(a, b): \text{both a and b are either odd or even}\} \). Show that R is an equivalence relation. Hence, find the elements of equivalence class [1].

[5 marks]

3. Check whether the relation S in the set of real numbers R defined by \( S = \{(a, b): \text{where } a - b + \sqrt{2} \text{ is an irrational number}\} \) is reflexive, symmetric or transitive.

[5 marks]

4. If N denotes the set of all natural numbers and R is the relation on \( N \times N \) defined by (a, b) R (c, d), if ad(b + c) = bc(a + d). Show that R is an equivalence relation.

[5 marks]
