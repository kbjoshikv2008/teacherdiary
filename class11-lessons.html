Chapter 1: Sets

Concepts: Introduction to Sets, Types of Sets, Operations on Sets, Venn Diagrams

Grade Subject Chapter Concept Total number of days

2. Types of Sets
3. Operations on Sets
4. Venn Diagrams		 8
Date Begins on Date No. of periods
01/04/2025 01/04/2025 10

Learning Outcomes:

Students will be able to:

  • Define sets and represent them in different forms (roster, set-builder).
  • Identify and classify types of sets (empty, finite, infinite, equal, equivalent, subsets).
  • Perform operations on sets (union, intersection, difference, complement).
  • Apply Venn diagrams to visualize set operations and solve problems.

Resources Required:

  • GeoGebra (for interactive Venn diagrams)
  • NCERT Activity Kit
  • Textbook: NCERT Class 11 Mathematics
  • Reference Book: CBSE Exemplar

Lesson Breakdown:

Objective Teaching Methodology Skills and Attitude Competencies
Define sets and represent them. Warm-up: Ask students to list objects in their school bags and group them. Introduction to roster and set-builder notation with examples (e.g., sets of even numbers, vowels). Logical Reasoning Conceptual Understanding
Classify types of sets (empty, finite, infinite, subsets). Visual Aids: Use charts to compare finite vs. infinite sets. Case Study: Real-life examples like "set of planets" (finite) vs. "set of stars" (infinite). Analytical Thinking Classification
Perform operations on sets (union, intersection, difference, complement). Hands-on Activity: Use colored circles to demonstrate union/intersection. Problem-Solving: Solve NCERT examples step-by-step. Problem-Solving Application of Operations
Apply Venn diagrams to solve problems. GeoGebra Demo: Interactive Venn diagrams for overlapping sets. Group Task: Solve problems like "In a class of 50 students, 20 play cricket, 25 play football..." Visual Interpretation Critical Thinking

Key Concepts:

  1. Sets: Collection of well-defined objects.
    • Roster Form: A = {1, 2, 3}.
    • Set-Builder Form: A = {x : x is a natural number ≤ 3}.
  2. Types of Sets:
    • Empty Set: ∅ = {}.
    • Subset: B ⊆ A if every element of B is in A.
  3. Operations:
    • Union (A ∪ B): All elements in A or B.
    • Intersection (A ∩ B): Common elements in A and B.
  4. Venn Diagrams: Visual tool for set relationships.

Case-Based Question:

A survey in a class of 40 students found:

  • 15 students like Maths.
  • 20 students like Science.
  • 5 like both.

Questions:

  1. Represent this data in a Venn diagram.
  2. Find the number of students who like only Maths.
  3. How many students like neither subject?

Assessment:

  1. Write the set {2, 4, 6, 8} in set-builder form.
  2. If A = {1, 2, 3} and B = {3, 4, 5}, find A ∩ B and A ∪ B.
  3. Prove that A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) using Venn diagrams.
  4. Let U = {1, 2, ..., 10}, A = {2, 4, 6}. Find A' (complement of A).

Learning Outcomes:

Students will be able to:

  • Understand and represent Cartesian products of sets
  • Define and identify different types of relations
  • Determine whether a relation is reflexive, symmetric or transitive
  • Understand the concept of functions and their types
  • Differentiate between relations and functions

Resources Required:

  • GeoGebra (for graphical representations)
  • Graph papers and coordinate grids
  • NCERT Activity Kit
  • Textbook: NCERT Class 11 Mathematics
  • Reference Book: CBSE Exemplar

Lesson Breakdown:

Objective Teaching Methodology Skills and Attitude Competencies
Understand Cartesian Products Warm-up: Use real-life examples like shirt-pant combinations. Visual demonstration using grid paper to plot ordered pairs. Group activity to create product sets. Logical Reasoning Conceptual Understanding
Define and represent relations Case Study: Use student height-weight data to show relations. Arrow diagrams to represent relations. GeoGebra visualization of relations on coordinate plane. Analytical Thinking Representation Skills
Identify types of relations Interactive activity: Students classify given relations. Think-Pair-Share: Examples from daily life (family relations, social media connections). Critical Thinking Classification
Understand functions and their types Hands-on: Use function machines to demonstrate input-output. Graphical method to identify one-one and onto functions. Real-world examples like temperature conversion. Problem-Solving Application

Key Concepts:

  1. Cartesian Product:
    • A × B = {(a,b): a ∈ A, b ∈ B}
    • Number of elements: n(A × B) = n(A) × n(B)
  2. Relations:
    • A relation R from A to B is a subset of A × B
    • Types: Empty, Universal, Identity
  3. Types of Relations:
    • Reflexive: (a,a) ∈ R ∀ a ∈ A
    • Symmetric: (a,b) ∈ R ⇒ (b,a) ∈ R
    • Transitive: (a,b), (b,c) ∈ R ⇒ (a,c) ∈ R
  4. Functions:
    • A relation where each input has exactly one output
    • Types: One-one (Injective), Onto (Surjective), Bijective

Case-Based Question:

In a school competition, students can participate in multiple events. Let:

  • A = {1, 2, 3, 4} represent 4 students
  • B = {Cricket, Chess, Debate} represent 3 events
  • R = {(1,Cricket), (2,Chess), (2,Debate), (3,Chess), (4,Cricket)}

Questions:

  1. Represent this relation using an arrow diagram.
  2. Is this relation a function? Justify your answer.
  3. Find the domain and range of R.
  4. If another student joins and only participates in Chess, how does R change?

Assessment:

  1. If A = {1,2} and B = {3,4}, find A × B and B × A. Are they equal?
  2. Let R be the relation on the set N of natural numbers defined by R = {(x,y): y = x + 5}. Write R in roster form.
  3. Check whether the relation R in R (real numbers) defined by R = {(a,b): a ≤ b} is reflexive, symmetric or transitive.
  4. Let f: R → R be defined by f(x) = x². Is f one-one? Is f onto?
  5. Let A = {1,2,3,...,14}. Define a relation R from A to A by R = {(x,y): 3x - y = 0}. Write its domain and range.

Learning Outcomes:

Students will be able to:

  • Convert between degree and radian measures of angles
  • Define trigonometric functions using unit circles and right triangles
  • Apply trigonometric identities to simplify expressions
  • Solve trigonometric equations using algebraic and graphical methods
  • Sketch and analyze graphs of trigonometric functions
  • Solve real-world problems involving periodic phenomena

Resources Required:

  • Unit circle charts and protractors
  • GeoGebra for dynamic visualizations
  • Graph papers and coordinate grids
  • NCERT Class 11 Mathematics textbook
  • CBSE Exemplar problems
  • Physical models (strings, circular discs)

Lesson Breakdown:

Objective Teaching Methodology Skills and Attitude Competencies
Understand angle measurement and conversion Hands-on: Measure classroom angles with protractors. Activity: Use strings on circular discs to demonstrate radians. GeoGebra simulations for angle conversion. Quantitative Reasoning Measurement Conversion
Define trig functions and identities Case Study: Model tides using sine functions. Group Activity: Derive identities geometrically (e.g., sin²θ + cos²θ = 1 using unit circles). Analytical Thinking Algebraic Manipulation
Apply sum/difference formulas Think-Pair-Share: Derive cos(x+y) geometrically. Problem-solving: Calculate exact values (e.g., sin15°, tan75°). Real-world examples: Surveying applications. Logical Deduction Formula Application
Graph trig functions Hands-on: Plot graphs using tables of values. GeoGebra sliders to explore amplitude, period, and phase shifts. Real-world connection: Sound wave patterns. Visual Representation Graphical Analysis
Solve trig equations Puzzle: Match equations to graphs. Case Study: Ferris wheel height equations. Error analysis: Identify and correct solution mistakes. Critical Thinking Problem Solving

Key Concepts:

  1. Angle Measurement
    • Radians: 180° = π radians
    • Arc length: l = rθ (θ in radians)
    • Conversions: Degree ⇄ Radian
  2. Trigonometric Functions
    • Unit circle definitions: sin θ = y/r, cos θ = x/r
    • Signs in quadrants: CAST rule
    • Fundamental identity: sin²θ + cos²θ = 1
  3. Sum/Difference Formulas
    • sin(x±y) = sin x cos y ± cos x sin y
    • cos(x±y) = cos x cos y ∓ sin x sin y
    • tan(x±y) = (tan x ± tan y)/(1 ∓ tan x tan y)
  4. Multiple Angles
    • Double angle: sin 2x = 2 sin x cos x, cos 2x = cos²x - sin²x
    • Triple angle: sin 3x = 3 sin x - 4 sin³x
    • Half angle: sin(x/2) = ±√[(1 - cos x)/2]
  5. Graphs
    • Periodicity: sin(x + 2π) = sin x
    • Key features: Amplitude, period, phase shift
    • Transformations: y = a sin(bx + c) + d

Case-Based Question:

Ferris Wheel Dynamics: A Ferris wheel has radius 20m and rotates at 2 radians per minute. At t=0, a seat is at the lowest point (ground level).

Questions:

  1. Model height h(t) above ground
  2. Find height after π/4 minutes
  3. When does seat first reach maximum height?

Solution Approach:

  1. h(t) = 20 - 20 cos(2t)
  2. At t = π/4: h = 20 - 20 cos(π/2) = 20m
  3. Max height when cos(2t) = -1 ⇒ t = π/2 minutes

Assessment:

  1. Convert 120° to radians and 3π/4 radians to degrees
  2. If sin θ = 3/5 and θ is acute, find cos θ and tan θ
  3. Prove: (tan θ + sec θ - 1)/(tan θ - sec θ + 1) = (1 + sin θ)/cos θ
  4. Sketch y = 2 sin(x - π/4) for x ∈ [0, 2π]. State period and amplitude
  5. Solve: 2 cos²θ + 3 sin θ = 0 for θ ∈ [0, 2π]
  6. Prove: cos²x + cos²(x + π/3) + cos²(x - π/3) = 3/2
Problem Solving Mathematical Modeling Graphical Analysis
